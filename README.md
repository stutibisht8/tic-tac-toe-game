# Tic Tac Toe Game (C Language)

This is a simple Tic Tac Toe game developed in C.

## 🎮 How to Play
- Player 1: X
- Player 2: O
- Enter positions from 1 to 9

## ▶️ How to Run
1. Compile the code:
   gcc tic_tac_toe.c -o game
2. Run the program:
   game

## 📌 Features
- 2 Player game
- Simple CLI interface
- Win & Draw detection
